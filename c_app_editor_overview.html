QRadar App Editor

Develop and edit apps directly in IBM® QRadar® by using the QRadar App Editor.

The QRadar App Editor is an easy-to-use editor that you start from the QRadar Admin tab. Any app developer can use the QRadar App Editor.

You can do any of the following tasks with the QRadar App Editor:
  • Develop new apps from a template.
  • Add, edit, and remove files and folders.
  • Delete apps that you create with the App Editor.
  • Import and edit existing apps.
  • Clone an app from a Git repository.
  • Deploy apps that you are editing and developing.
  • Export apps.
